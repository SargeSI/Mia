# Flybar vs FBL: Физика стабилизации

> ⚠️ В этом разделе были ДВЕ моих ошибки, исправленные Сергеем. Не повторять.

## Как работает flybar (ПРАВИЛЬНО)

Flybar стабилизирует **аэродинамически**, через циклический шаг лопастей, НЕ через изгибные моменты на валу:

1. Вертолёт кренится
2. Flybar-диск по инерции держит плоскость
3. Относительное движение → тяги → mixing lever → тарелка → **циклический шаг лопастей**
4. Аэродинамический восстанавливающий момент возвращает машину

**Ошибка Мии (исправлена):** «Изгибающий момент на валу → подшипники → корпус → рама». Это НЕВЕРНО. Flybar не давит на вал механически. Он меняет шаг лопастей.

## FBL-голова ≠ FBL-контроллер (важное различение)

- **FBL-голова** — механика: упрощённый хаб без flybar, без mixing lever. Тупая железка. Не стабилизирует.
- **FBL-контроллер** — электроника: BeastX, VBar, Spirit. В разрезе тракта приёмник→сервы. Читает IMU, добавляет поправки.

**Ошибка Мии (исправлена):** путала голову и контроллер как одно целое. Предлагала «гибрид flybar + FBL» — технически невозможный.

## Почему FBL-голова без контроллера неуправляема

FBL-голова статически неустойчива: убрали flybar → убрали возвращающую силу. Любое возмущение уходит в раскачку. Хвостовой гироскоп не помогает — он только в канале yaw.

## Почему Lama V4 летала без flybar

Lama V4 — **Fixed Pitch**, не Collective Pitch. Гибкие пластиковые лопасти при вращении образуют конус. При крене набегающий поток создаёт восстанавливающий момент — без всякого flybar. На CP с жёсткими лопастями конуса нет → естественной устойчивости нет.

**Вывод для нашего проекта:** CP с жёсткими лопастями требует ЛИБО flybar, ЛИБО FBL-контроллер. Одно из двух обязательно.

## Почему Ми-8 летает без flybar

Масштаб: лопасть 80 кг (против 25 г у T-Rex 450). Инерция диска — гироскоп сам по себе. Дополнительно: флаппинг-шарниры, delta-3, гидродемпферы, маятниковость (ЦМ ниже ротора). Совсем другая физика.

## Сигнальный тракт (архитектура)

```
Flybar:
ПРИЁМНИК → СЕРВЫ → ТАРЕЛКА → MIXING LEVER → ЛОПАСТИ
                              ↑
                          FLYBAR (механика, параллельно)

FBL:
ПРИЁМНИК → FBL-КОНТРОЛЛЕР → СЕРВЫ → ТАРЕЛКА → ЛОПАСТИ
              ↑
           IMU (гироскопы)
```

Flybar не в разрезе — отказ электроники не разрывает цепь. FBL в разрезе — отказ = потеря канала.

## МОС2-выводы

- Flybar: механический, не может отказать. МОС2 без вопросов.
- FBL-контроллер: софт + железо, требует DAL A. Не сертифицируем.
- FBL-голова без контроллера: статически неустойчива, неработоспособна.
