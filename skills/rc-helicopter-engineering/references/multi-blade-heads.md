# Multi-blade Rotor Heads

## Источник

Гайд на Hobby Squawk: "An Incomplete Guide to Multi-blade Main/Tail Rotor heads (450-800 size)"

Охватывает: CopterX, JDHMBD, Align, Gaui, Roban, Double Cats, PSG Dynamics, SK-Rotorkopf.

## JDHMBD 3-blade (450 класс)

- Цена: ~$72 (AliExpress)
- Main shaft: 5 мм
- 3/4/5 лопастные варианты
- "6CH 3D" на Banggood — **маркетинг.** Означает только «collective pitch с inverted», не жёсткий 3D-пилотаж
- Материал: авиационный алюминий, НЕ термообработанный
- Класс: scale-голова, для висения и спокойного полёта
- Отверстие в хабе: посадочное под shaft (∅5 мм), НЕ сквозное для pushrod
- Крепление: болт сверху
- Лопасти: пластик 315 мм, 29 г, mounting hole ∅3 мм

**Вывод:** для соосника не подходит — ∅5 мм занято shaft'ом, через него pushrod не провести.

## Gaui X3 3-blade (450 класс)

- Main shaft: 6 мм
- Производитель: Gaui (Тайвань)
- Качество: заводское, термообработанный алюминий, 3D-rated
- Цена: ~$80-100

## Align 550/600 3-blade (550-600 класс)

- Main shaft: 10 мм (совпадает с внешним валом для 600 класса)
- Производитель: Align (Тайвань)
- Качество: топовое, термообработанный алюминий, 3D-rated
- Цена: ~$120-150 за голову
- Потенциально: сквозное отверстие в хабе (классическая FBL, не DFC)
- **Наиболее перспективный вариант для 600 класса**

## Прочие поставщики

- **Roban** (600-800 класс): 10/12 мм shaft, 3/4/5-blade. Scale-качество.
- **SK-Rotorkopf.de**: articulated/rigid головы для 600, 3/4/5/6 blade
- **Double Cats**: БИЗНЕС ПРИОСТАНОВЛЕН
- **PSG Dynamics**: НЕ продают в США/Канаду
- **Aeropanda**: scale-головы СНЯТЫ С ПРОДАЖИ

## Важное различение

- **Scale-головы** (JDHMBD, CopterX, Roban): мягкий алюминий, низкие RPM, не для 3D. Подходят для висения.
- **3D-головы** (Align, Gaui): термообработанный алюминий, высокие RPM, жёсткий пилотаж. Подходят под нагрузки соосника.
- **DFC-головы**: безсквозные — не подходят для центрального pushrod.
- **Классические FBL**: потенциально имеют зазор в хабе — можно сверлить под pushrod.
