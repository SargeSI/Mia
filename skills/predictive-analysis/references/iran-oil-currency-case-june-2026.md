# Case Study: Iran Oil + USD/RUB + CNY/RUB — June 2026

Полный цикл анализа, проведённого в сессии 10-16 июня 2026.

## PIR
«Какова вероятность значительного роста Brent (>$85) на фоне конфликта с Ираном в ближайшие 30/90 дней?»

## Ключевые находки

### Исходный анализ (12 июня)
- Brent $83-90 (откат от пика $138 в апреле)
- OVX 54 (экстремальный страх)
- Конфликт активный: Operation Epic Fury (февраль), Hormuz закрыт, 17+ судов атаковано
- Дипломатический трек: Трамп заявил о прогрессе 11 июня
- ACH: H2 (Brent $75-95, статус-кво) = 60%, H1 (>$100, эскалация) = 30%, H3 (<$75, деэскалация) = 10%
- **Питфол: status-quo bias.** H2 доминирует формально (все evidence consistent). Применён фальсификационный подход + качественная корректировка.

### Review (16 июня) — Кримсон
- Снята блокада с иранского нефтеэкспорта
- ОАЭ разморозили $6-24 млрд КСИР
- MoU на 60 дней переговоров
- Ключевой инсайт от CEO нефтяных гигантов: открытие Ормуза ≠ обвал нефти
- **Трёхгоризонтный анализ:** июнь (шок вниз), июль-сентябрь (восстановление потоков + пополнение запасов = рост), октябрь-декабрь (новая норма с премией за риск)
- Пересмотр: H2 = 55%, H3 = 25%, H1 = 20%

### Бюджетная проекция
- Бюджет РФ-2026: Urals $71 × курс 92 = 6 532 руб/барр (план)
- Факт: Urals $78 × курс 70 = 5 460 руб/барр (−16.4%)
- Дефицит за январь-апрель: 5.88 трлн руб (уже превысил годовой план 3.79)
- При открытии Ормуза и Brent $70-75 — Urals $60-65, курс 70 = 4 200-4 550 руб/барр (−35% от плана)

### Bridge to Position
Позиция: USD/RUB (17 контр., вход 74.5), CNY/RUB (95 контр., вход 10.9), нефть Brent (5→2 контр., вход $95).

**Варианты:**
1. Кэш + ждать дна → перезайти (E = +2.06, риск забора 20%)
2. Шорт ближний + переворот (E = +10.03, риск gap up 6%)
3. Держать + докупать (E = −0.10, нет риска забора)
4. Сократить часть + докупать на дне (E = +1.25 при 70%, риск забора 0%)

**Выбор:** сократили 3 из 5 нефтяных контрактов (60% ≈ вариант 4). Остались в рынке, снизили просадку.

**Валютная часть:**
- USD/RUB: R/R докупки = 4.3 (спред входа 74.5 vs рынка 72.5 — 2 рубля)
- CNY/RUB: R/R докупки = 1.74 (спред входа 10.9 vs рынка 10.65 — 0.25)
- Баланс 42/34 USD/CNY — без перекоса
- При 92 руб/$: +559 000 — уверенный плюс

## Извлечённые уроки
1. **Три горизонта — обязательно.** Нельзя смешивать evidence разных временных масштабов в одной ACH-матрице
2. **Бюджетная проекция — стандартный follow-up.** Любой нефтяной PIR → проверить Urals × курс vs плановые параметры бюджета
3. **Бюджетное правило проциклично.** При падении нефти Минфин продаёт валюту → рубль укрепляется → нефтегазовые доходы падают ещё сильнее
4. **Status-quo bias.** H2 (статус-кво) всегда доминирует формально — применять фальсификационный тест
5. **Принципы риск-менеджмента для позиций:** (а) не сжечь депозит, (б) не стоять против рынка, (в) не сидеть с лосями на просадке. В этом порядке.
6. **Докупка консервативна:** +10-20%, не +50%. Цель — усреднение, а не максимизация экспозиции
