# Position Analysis: наложение сценариев на позиции

Методология расширения predictive-analysis для анализа конкретных торговых позиций.

## Когда применять

После построения ACH-матрицы и оценки вероятностей — когда нужно понять как сценарии отразятся на конкретной позиции.

## Протокол

### Шаг 1. Параметры позиции

Зафиксировать:
- Инструмент, направление, цена входа, текущая цена
- Экспирация (если есть)
- Размер экспозиции
- Плавающий P&L

### Шаг 2. Сценарные траектории

Для каждого сценария (H1/H2/H3) определить НЕ точечную цель, а **траекторию**:
- Краткосрочное движение (реакция на триггер)
- Среднесрочное (недели–месяцы)
- Конечная цель на горизонте

Это критически важно: одна и та же конечная цель может быть достигнута через просадку или без неё. Для позиции с экспирацией траектория важнее конечной цели.

### Шаг 3. P&L на траекторию

Для каждого сценария посчитать P&L на каждом участке траектории. Учесть:
- Если позиция с экспирацией — может не дожить до конечной цели
- Если вечный контракт (автопролонгация) — горизонт не ограничен, но есть фандинг
- Ролловер (для квартальных фьючерсов) — стоимость переноса

### Шаг 4. Математическое ожидание

E[P&L] = Σ P(сценарий) × P&L(сценарий)

**НО:** среднее — не главное. Важнее:
- Худший исход и его вероятность
- Риск маржин-колла
- Риск «забора» (выйти из позиции и не зайти обратно)

### Шаг 5. Выбор действия

Формировать варианты действий (закрыть, держать, докупить, сократить, хеджировать) и оценивать каждый через ту же матрицу сценариев.

**Критерии выбора (в порядке приоритета):**
1. Не сжечь депозит (худший исход)
2. Не остаться на заборе (gap risk)
3. Не тащить лося на просадке (нервы)
4. Максимизировать E[P&L]

## Питфолы

- **Одна цель ≠ одна траектория.** $95 к декабрю может быть достигнуто через $70 или через $90. Это принципиально разные исходы для позиции.
- **Gap risk.** Геополитические события могут вызвать разрыв цены — моделировать gap up/down отдельно от плавного движения.
- **Бэквордация/контанго.** Для фьючерсов форма кривой меняет P&L на ролловере.
- **Корреляция сценариев с позицией.** Позиция может быть сконструирована так, что все компоненты страдают в одном сценарии (например лонг нефти + лонг валюты при открытии Ормуза).
