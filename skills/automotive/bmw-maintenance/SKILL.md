# BMW Maintenance (E39, E53, E46, E38)

Навык для обслуживания и диагностики BMW конца 90-х — середины 2000-х.

## Принципы

1. **Точность в названиях компонентов.** Не смешивать системы:
   - **MK3/MK4** — навигационный компьютер (багажник), только картинка на монитор, звук через него НЕ идёт
   - **BM53/BM54** — радио-модуль (багажник), принимает аналоговый AUX, S/PDIF, управляет DSP
   - **16:9 монитор** — дисплей в салоне, жгут AUX примерно в 10 см от монитора
   - **DSP S677** — цифровой процессор звука (Top Hi-Fi), работает в паре с BM54

2. **Проверять OE-поставщиков.** Для BMW E-серии ключевые:
   - Подвеска: Lemförder (ZF Group) — OE для большинства рычагов
   - TRW — OE для отдельных моделей (E9X), не всегда для E53
   - Проверять кросс-ссылки Lemförder↔BMW через каталоги FCP Euro, Turner Motorsport, AutohausAZ

3. **Шины для тяжёлых SUV:** проверять индекс нагрузки и XL:
   - Для X5 4.4i минимальный индекс на ось считается по максимальной нагрузке ~1350 кг (перед) → 675 кг на колесо
   - 107 (975 кг) — норм для переда, 111 (1090 кг) — для зада с запасом
   - XL (Extra Load) — обязательно для тяжёлого SUV; отсутствие XL = мягкая боковина, риск грыжи

## Подбор шин — методология

1. Определить лето/зима/всесезонка
2. Для разношироких: проверить оба размера в выбранной модели
3. EU-лейблы: wet grip (A-E), fuel (A-E), dB. Для зимы — смотреть тесты на льду
4. Проверить OE-контракты: если шина шла на конвейер премиум-бренда — плюс к доверию
5. Проверить маркетинговые расхождения: all-season на Западе может продаваться как «летняя» в РФ
6. Сравнить цены в пределах размера, проверить XL

## Известные проблемы E53

### Биение тормозных дисков → разбитые шаровые рычагов

**Симптом:** новые диски бьют через ~1000 км трассы, даже при перебранных суппортах и новых направляющих. Шаровые опоры передних поперечных рычагов (31126760275/0276) разбиваются за 1-2 года.

**Цепочка:** бьющие диски → осевая вибрация ступицы → ударные нагрузки на шаровую опору → люфт → рычаг под замену. Сайлентблоки при этом живые (осевая нагрузка идёт не через них).

**Корень:** биение посадочного фланца ступицы. Допуск — 0.02 мм (проверка индикатором часового типа ИЧ-10). При бОльшем биении диск садится с перекосом → на трассе греется неравномерно → ведёт.

**Порядок устранения:**
1. Замерить биение фланца ступицы (индикатор 0.01 мм, магнитная стойка)
2. Если >0.02 мм — замена ступицы
3. Новые диски + новые рычаги Lemförder
4. Затяжка рычагов под нагрузкой (машина на колёсах)

### AUX — слабые басы на DSP S677

**Решение:** поднять AUX gain в сервисном режиме BM54:
- Для E53 2003-2004 с 16:9: зажигание 1 → радио ON → удерживать SELECT ≥8 сек → TONE LINE → пресеты #1-6
- Альтернатива: NCS Expert, кодирование модуля RAD

### BlueBus и CD-чейнджер

BlueBus ($199) эмулирует CD-чейнджер по I-Bus. При установке штатный чейнджер ОТКЛЮЧАЕТСЯ (BlueBus занимает его разъёмы питания и аудио). Это замена, а не дополнение. Для X5 с полным набором опций (DSP, TV, телефон, CD-чейнджер) BlueBus не подходит, если чейнджер нужен.

## Сервисные режимы

### BM53/BM54 с 16:9 монитором

| Год | Вход |
|-----|------|
| До 09/2001 | INFO → RDS → удерживать правую ручку ≥8 сек |
| 09/2001–03/2002 | Удерживать SELECT ≥8 сек |
| 2003+ | Удерживать SELECT ≥8 сек (как 09/2001+) |
