# Архитектура аудиосистемы BMW E53 с DSP S677

## Состав системы

BMW E53 с навигацией 16:9 и DSP Top HiFi (опция S677) — компоненты в багажнике:

```
                    ┌──────────┐
                    │ Монитор  │ ← Передняя панель
                    │  16:9    │
                    └────┬─────┘
                         │ AUX-разъём (~10 см от монитора, в жгуте)
                         │
    ┌────────────────────┼──────────────────────┐
    │  БАГАЖНИК          │                      │
    │                    ▼                      │
    │  ┌──────────┐  ┌──────────┐  ┌──────────┐│
    │  │   MK4    │  │   BM54   │  │   DSP    ││
    │  │ (NAV)    │  │  (RAD)   │──▶ (S677)   ││
    │  │          │  │          │  │          ││
    │  │ видео на │  │ коммута- │  │ цифровая ││
    │  │ монитор  │  │ ция      │  │ обработка││
    │  └──────────┘  │ аудио    │  └────┬─────┘│
    │                │          │       │      │
    │                │ AUX in   │       ▼      │
    │                │ CD/I-Bus │  ┌──────────┐│
    │                │ S/PDIF in│  │ Усилитель││
    │                └──────────┘  │ (аналог) ││
    │                              └────┬─────┘│
    │                                   │      │
    └───────────────────────────────────┼──────┘
                                        ▼
                                   Динамики
```

## Модули

### MK4 (NAV — навигационный компьютер)

- **Функция:** хранение карт, расчёт маршрута, вывод видео на монитор 16:9
- **НЕ участвует в аудиотракте** — ни AUX, ни обработка звука через MK4 не идут
- Стоит в левой нише багажника
- Частая ошибка: приписывание AUX-настроек к MK4. Настройки AUX — в BM54

### BM54 (RAD — радио-модуль)

- **Функция:** центральный коммутатор всех аудиоисточников
- Аналоговые входы: радио (AM/FM), AUX, телефон, CD-чейнджер (аналоговый вариант)
- Цифровой вход: S/PDIF от CD-чейнджера или BlueBus (для DSP-систем)
- Управление DSP: посылает цифровой сигнал в DSP-модуль
- **AUX gain** настраивается здесь (сервисное меню или NCS Expert → модуль RAD)
- Стоит в левой нише багажника, рядом с MK4

### DSP S677 (HiFi Professional)

- Цифровая обработка: кроссоверы, эквалайзер, временные задержки
- Принимает цифровой сигнал от BM54
- Выходит на аналоговый усилитель (или встроенный — зависит от комплектации)

## Пути сигнала

### AUX (аналоговый)

```
Телефон/плеер → BT-приёмник → 3.5mm разъём в жгуте монитора 16:9
→ аналоговый провод до BM54 (багажник)
→ BM54 AUX input (аналог)
→ BM54 → DSP (цифра)
→ DSP → усилитель → динамики
```

Слабые места:
- BT-приёмник (дешёвый ЦАП, низкий выходной уровень)
- Длинный аналоговый тракт от монитора до багажника
- AUX gain в BM54 (часто занижен с завода)

### CD-чейнджер / BlueBus (цифра, через I-Bus)

```
BlueBus (эмуляция чейнджера по I-Bus протоколу)
→ разъёмы питания и аудио чейнджера (в багажнике, за перегородкой)
→ BM54: I-Bus (управление) + S/PDIF (звук) или аналог (без DSP)
→ BM54 → DSP (цифра)
→ DSP → усилитель → динамики
```

Преимущества перед AUX:
- Цифровой тракт — нет потерь качества
- I-Bus управление — кнопки руля, метаданные на дисплее
- Питание от штатной проводки чейнджера — чистое, без помех

## Опции, связанные с аудио

| Код | Описание |
|---|---|
| S677 | HiFi Professional DSP — цифровая обработка звука (установлена на машине Сергея) |
| S638 | Car telephone Professional — штатный телефон Motorola с Bluetooth (установлен) |
| S601 | TV function — аналоговый ТВ-тюнер (установлен) |
| S672 | CD changer for 6 CDs — штатный чейнджер (установлен) |
| S609 | Navigation system Professional — MK4 (установлен) |
