# BlueBus — подробные спецификации

## Что такое BlueBus

Bluetooth-адаптер с полной интеграцией в I-Bus шину BMW. Разработчик — Тед Салмон (Ted Salmon). Всё открытое: железо, софт, прошивка.

- **Цена:** $199 USD (доставка ~2 недели)
- **Сайт:** https://www.bluebus.dev
- **GitHub:** https://github.com/tedsalmon/BlueBus
- **Поддержка:** support@bluebus.dev

## Технические характеристики (v2, актуальная)

| Компонент | Модель |
|---|---|
| Bluetooth-модуль | Microchip **BM83** (замена BC127, ставшего unobtainium) |
| Протоколы BT | A2DP, HFP, SPP, BLE, MAP |
| S/PDIF трансивер | Texas Instruments **DIT4096** |
| ЦАП | TI **PCM5122** (2.1V RMS, 113dB SNR) |
| I-Bus трансивер | Melexis **TH3122.4** (аппаратное обнаружение коллизий) |
| Усилитель для звонков | PAM8406 |
| USB-UART | FT231X |
| Память | 1MB EEPROM |
| Разъёмы | TE Connectivity (тот же поставщик, что у BMW) |

## Возможности

- **A2DP** — стерео-аудио с телефона
- **HFP** — hands-free звонки (требуется телефонный адаптер)
- **AVRCP** — управление с кнопок руля, метаданные на дисплее
- **Эмуляция CD-чейнджера** — магнитола видит BlueBus как родной чейнджер по I-Bus
- **Эмуляция штатного телефона** — звонки работают независимо от режима магнитолы (радио/CD/etc)
- **S/PDIF** — цифровой аудиовыход для машин с DSP
- **Caller ID** — отображение номера на дисплее (v2: только номер, без имени — обещают прошивкой)
- **Comfort-фичи** (не на всех): one-touch поворотники, автозакрытие дверей по скорости, автооткрытие по ключу
- **OTA-обновления прошивки** через USB

## Ограничения v2 (по сравнению с v1)

- Caller ID: только номер звонящего, без friendly name (ждёт софтового фикса)
- Автоматическое управление громкостью **не работает** (включая приглушение при заднем ходе)
- AVRCP browsing (листание плейлистов с головы) — заявлено на будущее

## Телефонный адаптер

Докупается отдельно (~€35). Нужен только для HFP (звонки). Поддерживает штатный микрофон BMW. Три варианта переходников под разную проводку. При заказе требуется VIN и фото телефонного модуля.

## Установка (E39/E53)

1. Багажник: снять панель пола
2. Добраться до задней перегородки (между багажником и салоном)
3. Найти нижний вырез по центру перегородки
4. Опознать разъёмы питания и аудио CD-чейнджера
5. Вытащить разъёмы из чейнджера → вставить в BlueBus

**CD-чейнджер после этого не работает.** BlueBus физически занимает его разъёмы.
