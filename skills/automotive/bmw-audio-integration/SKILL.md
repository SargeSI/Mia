# BMW Audio Integration (E39/E53)

Знания по интеграции Bluetooth и AUX в старые BMW с I-Bus шиной: E39 (1995–2004), E53 (1999–2006). Загружать при вопросах Сергея по аудио/Bluetooth в его машинах.

## Архитектура аудиосистемы E39/E53

### Компоненты (все в багажнике)

| Модуль | Функция | Где стоит |
|---|---|---|
| **BM54** (RAD) | Радио-модуль, коммутация аудиоисточников, приём AUX, управление DSP | Багажник, левая/правая ниша |
| **MK4** (NAV) | Навигационный компьютер, карты, вывод видео на монитор 16:9 | Багажник, левая ниша |
| **DSP** (S677) | Цифровой процессор звука, кроссоверы, эквалайзер | Багажник, левая ниша |
| **CD-чейнджер** | I-Bus устройство, эмулируется BlueBus | Багажник |
| **Монитор 16:9** | Дисплей, видео с MK4, **AUX-разъём в жгуте монитора** (~10 см от монитора) | Передняя панель |

### Сигнальный тракт AUX

```
BT-приёмник → AUX-разъём (жгут монитора 16:9) → BM54 (аналоговый вход) → DSP → усилитель → динамики
```

**Критично:** BM54 ≠ MK4. MK4 — только навигация и видео. AUX gain настраивается в **BM54**, не в MK4.

### Сигнальный тракт CD-чейнджер / BlueBus

```
BlueBus (эмуляция чейнджера, I-Bus) → BM54 (S/PDIF или аналог) → DSP → усилитель
```

## Варианты Bluetooth-интеграции

### 1. AUX + BT-приёмник (колхоз, но сохраняет всё штатное)

**Плюсы:**
- Не трогает штатное оборудование
- Чейнджер, телефон, TV — всё остаётся работать
- Дёшево

**Минусы:**
- Аналоговый тракт — потери качества, слабые басы
- Нет управления с кнопок руля
- Нет метаданных на дисплее
- Требует качественного BT-приёмника (aptX/LDAC, выход ≥1V RMS)

**Улучшения:**
- Поднять **AUX gain в BM54** через сервисное меню или NCS Expert (кодирование модуля RAD)
- Заменить дешёвый BT-приёмник на нормальный (CSR8675, QCC5125)
- Питать приёмник от ACC с фильтром (не от прикуривателя — помехи по земле)

### 2. BlueBus (цифра, но теряется CD-чейнджер)

**Плюсы:**
- Цифровой звук (S/PDIF в BM54) — басы, чистота
- Полная интеграция: кнопки руля, метаданные на дисплее, Caller ID
- Эмуляция штатного телефона (HFP)
- Открытый исходный код

**Минусы:**
- **Вытесняет CD-чейнджер** — физически занимает его разъёмы питания и аудио на I-Bus. Это замена, не дополнение
- $199 + доставка
- v2: Caller ID только номер (без имени), нет автоуправления громкостью

**Установка:** E39/E53 — в багажнике, разъёмы чейнджера за перегородкой между сиденьями и багажником. Выдёргиваешь разъёмы из чейнджера → втыкаешь в BlueBus.

### 3. GROM Audio / Dice Mediabridge

Аналоги BlueBus — тоже эмулируют чейнджер, тоже вытесняют его. У некоторых GROM заявлен pass-through для чейнджера, но работает нестабильно.

## Подводные камни

1. **MK4 ≠ BM54.** Не путать навигационный компьютер с радио-модулем. AUX-тракт идёт через BM54, настройки gain — в BM54. MK4 только гонит картинку на монитор.

2. **AUX-разъём — в жгуте монитора 16:9**, примерно в 10 см от самого монитора. Это не отдельный разъём на BM54.

3. **BlueBus не сосуществует с CD-чейнджером.** Это либо-либо. Если сохранение всего штатного оборудования критично — только AUX + BT-приёмник.

4. **DSP S677** (HiFi Professional) — цифровая система. BlueBus может подать S/PDIF напрямую в BM54, минуя аналоговый тракт. С AUX-приёмником цифрового входа нет — только аналог.

5. **Машины до 1998 г.** могут быть на M-Bus, а не на I-Bus — BlueBus не заработает без переделки проводки.
