# BMW PA Soft (BMW Scanner 1.4) — справочник по кодированию

## ADFG — запись FGSTNR в модуль

**ADFG** (в разделе Reprogramming PA Soft) — функция записи производного от VIN в модуль.

**Формат:** 6 цифр — последние 6 ЦИФР VIN, буквы пропускаются.

**Пример:** VIN WBAFB51040LU50083:
- Все цифры VIN по порядку: 5 1 0 4 0 (L — буква, пропуск) (U — буква, пропуск) 5 0 0 8 3
- Последние 6 цифр: `050083`
- ADFG = `050083`

**Применяется:** телефонный модуль, LCM (свет), IKE (приборка — но там ADFG обычно FFFFFF, используется FGSTNR).

**Разночтения:** некоторые руководства PA Soft указывают 4 последние цифры (`00XXXX`), но поле ожидает 6 цифр. Если модуль не принимает значение из 4 цифр — использовать 6 цифр (последние 6 цифр VIN, исключая буквы).

**Если ADFG не совпадает с VIN:**
- Модуль может не видеться другими блоками
- Кнопки могут быть неактивны (например «Register new headset» в телефонном модуле)
- Исправить: выбрать модуль → Reprogramming → Write ADFG → ввести 6 цифр → OK

## ZKE — Selective central locking

**Selective central locking** (селективная блокировка): первое нажатие кнопки разблокировки открывает только водительскую дверь. Второе — все остальные.

**Кодирование:** ZKE → Coding → selective central locking → active / not active.

**Заводские настройки:**
- US-версии: active
- EU-версии: not active

## ZKE — DWA (штатная сигнализация)

**DWA** — Diebstahl Warn Anlage, штатная сигнализация. Отключается в ZKE → Coding → DWA → nicht_aktiv.

**Нюанс:** отключение через кодирование отключает сирену и аварийки, но сам блок DWA может продолжать мониторить датчики и мешать шине заснуть. При поиске утечки тока — физически отключить разъём модуля DWA.

**Индикатор:** красный диод на нижней части центрального зеркала. Мигает при постановке на охрану. Даже с отключённой DWA может кратко мигнуть при закрытии — это норма, просто подтверждение закрытия замков.

## Электрохромные зеркала E53

**Схема:** центральное зеркало — ведущее. На него приходит +12V (зажигание) и масса. Внутри: два фотосенсора, схема сравнения, собственный электрохромный элемент. На выходе — два провода на боковые зеркала с сигналом затемнения 1.5V.

**Логика затемнения:** датчик сзади (свет фар) ярче датчика спереди (освещённость снаружи) → подача 1.5V. Проверка: посветить фонариком в задний датчик, передний закрыть рукой.

**Не работает ни одно из трёх — общие причины:**
1. Предохранитель (в бардачке, часто общий с подсветкой салона)
2. Питание на разъёме центрального зеркала
3. Окисление контактов разъёма (особенно после колхоза)

**Колхоз с подсветкой ног:** разъём затемнения и разъём подсветки ног физически похожи. Если перепутаны — 12V от подсветки подаётся в линию затемнения 1.5V. Это выжигает выходной каскад центрального зеркала → центральное мертво → боковые без сигнала.

**Диагностика:** мультиметром на разъёме центрального зеркала. Если +12V есть, а затемнения при фонарике нет — электроника зеркала мертва. Менять центральное зеркало.

## Поиск утечки тока на E53

**Норма:** ≤ 50 мА через 16 минут после закрытия (время засыпания всех блоков).

**Метод:** мультиметр в разрыв минусовой клеммы АКБ, ждать 16 минут, далее вынимать предохранители по одному — где ток упал, тот потребитель и виноват.

**Типичные убийцы АКБ на E53:**
1. FSR (резистор печки) — даже с выключенным зажиганием
2. Телефонный модуль — особенно с битым ADFG
3. Монитор 16:9 / MK4 — подсветка или вентилятор не гаснут
4. Усилитель DSP — не уходит в сон
5. Блок DWA — мониторинг датчиков (решается физическим отключением разъёма)
6. Компрессор пневмы — если подключён нештатно, но штатный EHC должен засыпать
