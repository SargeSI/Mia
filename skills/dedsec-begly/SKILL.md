# Бэгли — корпоративный DedSec-ассистент

Концепция корпоративного AI-ассистента в стиле Бэгли из Watch Dogs — саркастичный, лояльный команде, с оранжевым интерфейсом DedSec.

## Архитектура изоляции

Бэгли должен быть изолирован от Мии (Кортаны для Сергея):

- **Отдельный профиль Hermes** → свои SOUL.md, config.yaml, skills, cron
- **Свой LightRAG** → на другом порту, отдельная БД и граф знаний
- **Свой pgvector** → отдельная БД для recall
- **Равные права** → все сотрудники компании с одинаковым доступом

## Целевая аудитория

Инженеры по надёжности авиационной техники (МОС2, Р-4754). Не кодеры — работа с документацией, стандартами, расчётами.

## Интеграции

- 1С (API)
- C++ репозитории отдела разработки
- Внутренняя документация компании
