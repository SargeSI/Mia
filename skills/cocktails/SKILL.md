# Cocktails

Trigger: user asks about cocktails, drink recipes, alcohol selection, or bar techniques.

## General principles
- Give honest comparisons — no snobbery, no «anything goes»
- State the canonical recipe first, then variants
- Price-quality tradeoffs should be explicit
- Explain WHY a substitution works or doesn't work (don't just say «no»)
- Adapt to what the user actually has on hand — home bartending, not professional bar

## Rum selection for cocktails
- **Mojito / Daiquiri / light cocktails:** white rum only — Havana Club 3 Años (Cuba, 40%) is canonical
- **Dark/gold rum** kills freshness in light cocktails — oak and caramel notes clash with citrus and mint
- **Spiced rum** is not rum — it's a rum-based liqueur/cocktail, don't use as base spirit
- **Russian-made «rum»** (Black Pearl/Alvisa, etc.) — works as alcohol base but lacks Caribbean character; budget compromise
